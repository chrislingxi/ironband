# Ironband 完整版 ~8 小时自主开发路线图

> 目标: 把第一幕打磨成一个**完整、可玩、有 D2 味道**的产品(深度优先, 而非铺更多幕)。
> 现状(已上线): 等距渲染+程序化精灵+光照 · 命中/抗性/受身/打击感 · 4 怪AI · 物品词缀掉落 ·
> 装备改战力 · 经验升级 · 第一幕9区域链+营地NPC+出口传送 · 三职业技能树数据+难度。

## 执行约定 (每个阶段)
1. 并行子 agent 产出互不重叠的新模块 → 我串行集成
2. `npm run typecheck && npm test && npm run build` 全绿, 关键逻辑配单测
3. 提交 → 推 `claude/diablo2-mobile-refactor-3jv8z3` → CI 绿 → 合并 main → 自动部署上线
4. 全程**原创文本 + 程序化美术/音频**, 版权素材不入库(资产覆盖管线已就位)
5. 完成阶段后在本文件勾选; 遇到需用户拍板的阻塞 → 停下简报

## 阶段 (按优先级)
- [ ] **A. 三职业可玩 + 职业选择** — 标题/选职界面; 投射物系统(箭/法术); 亚马逊(弓/标枪)与法师(冰火电)的3技能键实战化; 逐职业起手属性/装备/派生
- [x] **B. 战斗深度 + 怪物扩充** — 4新怪种(弓手/督军/吐毒虫/恶犬); 精英怪群(6词缀+描边名牌+必掉); 敌方投射物(萨满火球/弓手箭/毒); 死亡爆炸
- [x] **C. 安达莉尔 Boss + 任务系统** — 三阶段毒系Boss(近战毒爪/环形毒弹/召唤/狂暴, 暴掉装备); 任务链(清场触发完成)+ 任务日志UI(📜) + 奖励(技能点/雇佣兵解锁/通关)

- 深度优先: 第一幕做完整, 暂不开第二幕
