# 设计决策（已锁定）

## 目标
重构出一个**真正有 D2 味道**的等距刷宝 ARPG（手游/Web，纯单机）。旧版"不像 D2"的根因：
俯视几何渲染（等距素材画错了）、战斗无命中判定、刷宝深度不足。本项目逐一修复。

## 技术
- TypeScript + Vite + **PixiJS(WebGL)**；固定步长游戏循环；轻量 ECS。
- 数据驱动：D2 风格数据表（语义对齐 `.txt`，见 `src/game/data/schema.ts`）。
- 发布：GitHub Pages 静态站；存档 IndexedDB + 导出码。

## 忠实复刻 D2 的支柱（实现时对照 Arreat Summit / 公开数据）
- **等距 2.5D** + 深度排序 + 光照半径
- **命中判定**：`ToHit% = 100·AR/(AR+DR) · 2·alvl/(alvl+dlvl)`；硬直(FHR)；格挡(FBR)
- **伤害类型**：物理/火/冰/电/毒/魔法 + 抗性 + Hell 免疫
- **刷宝**：白/蓝/黄/套装/暗金；前后缀词缀（受 ilvl/mlvl 限制）；TreasureClass 掉落；宝石/插槽；赌博；合成方
- **构筑**：力敏体能精力；三系技能树×3职业；synergy；+技能；FCR/FHR/FBR 突破点
- **怪 AI 个性**：堕落者逃跑+集结、萨满复活、精英词缀包
- **难度**：Normal/NM/Hell

## 端游 → 手游 适配（已确认）
- **移动**：左侧浮动虚拟摇杆
- **技能**：自动攻击 + **3 个技能键**（技能树投点后绑定）
- **无药水系统**（移除腰带药水）
- **背包简化为单格装备**（所有装备占 1 格）
- **仓库 / 角色属性 / 技能树**：忠实复刻（全屏面板）
- **营地**：参考 D2（铁匠/Akara/Charsi/Gheed/雇佣兵/航海家 等服务）
- **QoL**：自动拾取金币、掉落过滤、航点快传、断点续玩
- 横屏 + 安全区适配；多怪性能用批渲染 + 对象池 + LOW_FX 自适应

## 职业 & 范围
野蛮人 → 亚马逊 → 法师（全做）。首切片：M0 等距能走动 + M1 野蛮人近战手感。
第一幕完整（营地 + 多区域 + 任务链 + 安达莉尔）。后续幕 / NM·Hell / 符文之语 / 套装。
