## B. 战斗深度 + 怪物扩充 (待确认)
- **精英怪群**: 每片野外随机 1–2 个精英队长 + 随从。队长随机带 1–2 条词缀:
  极速 / 火焰附魔(死亡爆炸) / 寒冷附魔(命中减速) / 闪电附魔 / 增益光环 / 坚韧(高血)。
  精英掉落更优(稀有率↑, 必掉若干)。
- **怪种扩充**: 现 4 种 → ~6–8 种(复用现有精灵+程序化色调): 加 弓箭手(远程) / 督军(高血近战) / 蛆虫等。
- **远程敌人投射物**: 萨满火球、弓箭手箭改为真飞射物(敌方 missile 打玩家, 可被走位躲)。
- **决策点**: 精英词缀池选哪些? 精英密度(默认 1–2/区域)? 怪种要不要这么多?

## C. 安达莉尔 Boss + 任务系统 (待确认)
- **安达莉尔(巢穴 Boss)**: 大体型, 三阶段血量: ①近战毒爪 ②召唤小怪+毒云 AOE ③狂暴(攻速↑)。
  毒系为主(凸显玩家毒抗)。击杀 → 通关 Act1 + 保底掉落(≥1 稀有/暗金)。
- **任务系统**: 任务状态机(未接/进行/完成) + 任务日志 UI(📜) + NPC 给/交任务。三任务:
  - Q1 净化邪恶巢穴: 清空 den_of_evil → 奖励(技能点 / +1 全抗)
  - Q2 姐妹的安息之地: burial_grounds 击败精英队长 → 解锁雇佣兵
  - Q3 击败安达莉尔: 通关 Act1
- **决策点**: Boss 机制取舍(三阶段够不够/要不要更花哨)? 任务奖励给什么?
