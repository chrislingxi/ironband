# 美术 / 音乐 资产方案

## 原则
仓库**只含原创代码 + 开源占位素材**。D2 原版美术/音乐是版权资产，**不入库、不分发**。
游戏开箱即用一套 D2 风格的开源素材；想要原版质感的玩家，用提取工具从**自有合法安装**本地覆盖。

## 两层资产
1. **开箱占位（入库）**：FLARE（Clint Bellanger，CC-BY-SA，本就是类 D2 等距 ARPG 素材）
   + 精选 CC0/CC-BY 哥特氛围素材与音乐。正确等距渲染下已有 D2 既视感。位置：`assets/`。
2. **原版覆盖（不入库，玩家本地）**：用 `pipeline/` 提取工具在自有 D2 安装上运行，
   解包 MPQ → DC6/DCC 精灵转 PNG 图集 + JSON 帧、DS1/DT1 地砖转瓦片、音频转 ogg →
   输出到 `assets/extracted/`（被 `.gitignore` 排除）。运行时优先加载，缺失回退占位。

## 加载契约
- 资产以 key 引用（数据表里的 `sprite` / `icon` 字段）。
- 加载器解析顺序：`assets/extracted/<key>` → `assets/<key>`（覆盖式回退）。
- 运行时只读 Web 友好的 PNG/JSON/ogg（不在浏览器里解 MPQ，手机性能好）。

## 边界
提取工具与格式规范由本项目提供；**运行提取、放置原版素材是玩家在自己机器上的个人行为**，
原始版权资产不进仓库、不经分发。这样既保证"开箱好看 + 可选原版质感"，又保持工程干净合规。
